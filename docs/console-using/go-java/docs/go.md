# Golang

This guide will start with the installation of the GenAuth Golang SDK and step by step guide you on how to quickly add user authentication and management capabilities to your existing or newly developed applications.

<AppDetailSiderBar />

## Installation

```
go get -u github.com/Authing/authing-golang-sdk
```

## Authenticate your user

### Initialization

```go
import (
"github.com/Authing/authing-golang-sdk/authentication"
)
func main() {
var authClient *authentication.Client
var err error
authClient, err = authentication.NewClient(&authentication.AuthenticationClientOptions{
AppId: "GEN_AUTH_APP_ID",//Application ID
AppSecret: "GEN_AUTH_SECRET",//Application Secret
Domain: "GEN_AUTH_DOMAIN",//User pool domain name corresponding to the application
RedirectUri: "GEN_AUTH_REDIRECTURI",//Redirect target URL after authentication is completed
LogoutRedirectUri: "GEN_AUTH_LOGOUTREDIRECTURI"//Logout callback address
Scope: "GEN_AUTH_SCOPE"// Permissions requested by the application side GenAuth
})
}
```

### Simple authentication user

```go
// Generate authentication address
result, err = authClient.BuildAuthUrl(&authentication.AuthURLParams{
State: "Context data generated by the requester, can be empty, users prevent cross-site attacks",

Nonce: "Context data generated by the requester, can be empty, will be included in the IDToken returned by the callback",
RedirectUri: "Callback address, can be empty, default to use the RedirectUri parameter in the NewClient function",
Forced: false,//Whether to force re-login, if true, Authing will force the user to re-login, default is false
})
```

## Manage your users

### Initialization

```go
options := client.ManagementClientOptions {
AccessKeyId: "GEN_AUTH_USERPOOL_ID", // User pool id
AccessKeySecret: "GEN_AUTH_USERPOOL_SECRET", // User pool key
}
client, err := client.NewClient(&options)
if err != nil {
panic(err)
}
```
